Project overview

Developed a web application to provide health care information to young adults in a team of five interns (shout out to the Bill Gates Fun Club! Rachel, Ian, Carlos,Denis)

My contribution

I implemented the budget filtering feature in our application that enabled a user filter doctors based on their cost and render the information of doctors whose average cost of services is less than or equal to the value a user inputs using the budget slider. To implement this feature, I used React redux which enabled global state management of a list of doctor objects in our React applicationâ€™s store. Each doctor is an object of the doctor class that takes in information like name, specialty, location, services offered, years in practice, and others. To enable real-time update of a filtered doctor list which is also a list of doctors objects, I created React redux method to filter and append all doctors whose average cost is less than or equal to the budget slider value to a filtered list also stored in our redux store. Since this filtered list is stored in the Redux store, then the list was globally accessible and enabled real time update of doctor information in our application as the user moved the budget slider.
