Search feature for Feeding Kentucky Database

For over 10 years, FeedingKY has not collected enough detailed information from farmers, especially about their main crops. Feeding KY needs to reformat its existing database to track farmers' information to track where grant money is being spent. I worked in a team of three to add a feature that allows a user to search and view results from the FeedingKy database website. I worked on the SQL queries and how the results from the query were passed on to the Google Chart API
